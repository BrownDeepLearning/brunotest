# brunotest-preview

A VSCode extension to seamlessly integrate and use `brunotest`

## Features

-   Live Preview
-   Run Brunotest via Command Palette
-   Create Stencil via Command Palette
-   Create Chaff via Command Palette
-   ...and more?

## Requirements

```bash
cd brunotest-preview
npm install
```
